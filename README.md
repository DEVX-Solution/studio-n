# Studio N. Fullstack

Projeto convertido do HTML enviado para **React + Node.js + Express + Tailwind CSS**.

## Como rodar

```bash
npm run install-all
npm run dev
```

Abra:

```txt
http://localhost:5173
```

Backend:

```txt
http://localhost:5000/api/health
```

## Funcionalidades

- Layout fiel ao Studio N.
- React componentizado
- Tailwind CSS
- Node.js + Express
- Formulário funcional abrindo e-mail com mensagem pronta
- Links de contato clicáveis
- Navegação com seção ativa
- Menu mobile
- Símbolos/setas trocados por ícones SVG
